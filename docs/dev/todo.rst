How to Help
===========

StackAPI is under active development, and contributions are more than welcome!
There are details at :ref:`Contributing <contributing>`, but the short version
is:

1. Check for open issues or open a fresh issue to start a discussion around a bug.
2. Fork the `repository <https://github.com/awegnergithub/stackapi>`_ on GitHub and start making your
   changes to a new branch.
3. Write a test which shows that the bug was fixed.
4. Send a pull request.
   Make sure to add yourself to `AUTHORS <https://github.com/awegnergithub/StackAPI/blob/master/AUTHORS.rst>`_.
